# Gidy Profile - Full Stack Replica

A full-stack replica of the Gidy profile page built using MERN stack with TypeScript.

## Tech Stack

- Frontend: React + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: MongoDB
- Styling: Tailwind CSS

## Project Structure

- client/ → Frontend application
- server/ → Backend API
- docs/ → Architecture & design notes
